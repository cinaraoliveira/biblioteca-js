# 📚 Biblioteca Digital

Sistema simples de biblioteca desenvolvido em JavaScript (front-end),
aplicando conceitos de Programação Orientada a Objetos.

## Conceitos aplicados
- Abstração
- Encapsulamento
- Herança
- Polimorfismo
- Agregação / Composição
- Tratamento de exceções

## Tecnologias
- HTML
- CSS
- JavaScript (ES6)

## Como executar
Basta abrir o arquivo `index.html` no navegador
ou utilizar Live Server no VS Code.
